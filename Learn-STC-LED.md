# STC15F2KS60-LED的闪烁

注意定义L1/2/3/4对应的引脚

## 代码块

“8421” 对应引脚，高低电平
0点亮
1熄灭   注意看原理图，分析工作原理。

```c
void led_1(void)
{
 P1=0xff;//1111 1111
 delay1s();
 P1=0x0f;//0000 1111
 delay1s();
 P1=0xf0;//1111 0000
 delay1s();
 P1=0xc0;//1100 0000
 delay1s();
 P1=0x80;//1000 0000
 delay1s();
 P1=0x80;//0000 0000
}
```

### 左移代码

P1左移一位

```c
P1=_crol_(P1,1);
```
